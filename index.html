Desplagabilidad

desplegabilidad es la facilidad de desplegar una aplicación en producción


Un escenario para la aplicacion de despliegue es el más común de las tácticas a la hora de desplegar un nuevo proyecto ya que este determina cómo se va a mostrar el sistema al publico


desplegabilidad es una táctica única y no tiene más tácticas que la componen

Disponibilidad

disponibilidad es la táctica que determina los periodos de tiempo que está disponible un sistema


los escenarios de disponibilidad son todos aquellos en los que se determina todo el estado del sistema, si se cayó o si toca restablecerlo


disponibilidad depende de factores como el detectar, recuperarse y prevenir fallas

Escalabilidad

escalabilidad es la cantidad que solicitudes que puede manejar un sistema exitosamente


un escenario donde se puede representar escalabilidad es un monitoreo de las cargas que recibe el sistema en su momento y se detectaría cualquier sobrecarga o inestabilidad


escalabilidad es una táctica única, no dispone de más tácticas

Interoperabilidad

interoperabilidad es la capacidad de un sistema de intercambiar información entre dos o más sistemas


en interoperabilidad se podría describir el constante monitoreo de los sistemas al momento de intercambiar información y se administra si la información se trasladó correctamente


interoperabilidad tiene factores de localización, gestionar interfaces y medir tiempos de respuesta

Modificabilidad

modificabilidad es la táctica de un sistema para aceptar cambios y de añadir funcionalidades


en modificabilidad se toma en cuenta cuando el sistema se añaden nuevas funciones y se maneja como va a tomar el cambio el sistema y si hay algun rango de error


modificabilidad usa tácticas como reducir módulos,aumentar cohesión y reducir acoplamientos

handyman

Rendimiento

rendimiento es el desempeño del sistema y los recursos que utiliza


en rendimiento se utilizaria en un escenario donde el se compruebe si un sistema soporto las diferentes pruebas de rendimiento y no presenta inestabilidades


rendimiento usa características como latencia, respuesta y control de recursos

Seguridad

La seguridad es una medida del sistema que se utiliza para proteger datos y otros tipos de información de sistema contra accesos no deseados


Este tipo de escenarios se pueden implementar en todo tipo de cuentas informáticas como correos electrónicos o cuentas bancarias y para proteger la información de estas se implementan


Las tácticas de seguridad se dividen en cuatro tipos: Detectar Ataques, Resistir ataques, Reaccionar a ataques, Recuperarse de los ataques

lock

Usabilidad

usabilidad es como un sistema interactúa con un usuario y cual es la facilidad de uso


en usabilidad se usaria en escenarios en los que el sistema tenga interacciones con el usuario como interfaces


usabilidad usa tácticas como cancelar, deshacer, pausar o reanudar y agregar y se describen como sistemas de apoyo y de soporte
